# 🌸 BloomWatch

**BloomWatch** is a responsive stopwatch web application with a calming nature-inspired design.

## ✨ Features

- ⏱️ Start, Pause, Resume & Reset
- 🌸 Bloom Lap recording
- 📊 Lap Statistics
- 📤 Export lap history
- 🕒 Live Date & Time
- 📱 Fully Responsive
- 🌿 Nature-inspired UI
- 💚 Comfortaa Font

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
BloomWatch/
│── index.html
│── style.css
│── script.js
│── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or run it using Live Server.
